# Multi-Weapons
```(formally Modal exotic weapons)```

Multi-Weapon have the capacity to switch between two or more weapons at will using an Object Interaction; such as a vibrostaff that can switch to a blaster rifle or a blaster rifle with a built in grenade launcher or sniper rifle.

### DC-17m Interchangeable Weapon System
*Multi-Weapon [(heavy carbine, sniper rifle, grenade launcher)](https://sw5e.com/loot/weapons/?search=Heavy%20carbine), <font style="color:lime">premium</font>*
10lb

*A type of reconfigurable repeating blaster rifle used by Republic clone commandos during the Clone Wars and the early days of the Empire.*

This Multi-Weapon comes with three modes based on the following weapons: a heavy carbine, a sniper rifle and a grenade launcher and you track the reload value for each mode separately. Only one mode can be active at a time. You can change which mode is active as an Object Interaction.
